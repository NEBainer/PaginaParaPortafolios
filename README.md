# Portafolio – Ezequiel Bainer

Este portafolio reúne mis proyectos y experiencias como desarrollador en formación, con foco en desarrollo web y backend.

No es solo una muestra de resultados finales, sino también del proceso de aprendizaje: cómo encaro problemas, cómo estructuro soluciones y cómo voy incorporando buenas prácticas de la industria.

---

## ¿Qué vas a encontrar acá?

- Proyectos personales orientados a backend y desarrollo web
- APIs REST desarrolladas con Java (Spring Boot) y Python (FastAPI)
- Ejemplos de arquitectura por capas, validaciones, manejo de errores y uso de bases de datos
- Un recorrido por las tecnologías que manejo actualmente

---

## Sobre los proyectos

Cada proyecto publicado en este portafolio busca resolver un problema concreto y está pensado como un entorno de aprendizaje realista.

No son proyectos “perfectos”, sino que reflejan cómo aprendo, cómo mejoro el código y cómo adopto herramientas utilizadas en entornos profesionales.

---

## Tecnologías principales

- Java / Spring Boot
- Python / FastAPI
- SQL / MongoDB
- HTML, CSS, JavaScript
- Git y GitHub
